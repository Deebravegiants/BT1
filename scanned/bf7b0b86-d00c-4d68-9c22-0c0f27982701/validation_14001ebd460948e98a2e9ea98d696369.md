### Title
Unprivileged Caller Can Trigger Protocol-Wide Auto-Pause via Public `updateRSETHPrice()` and Unguarded `RETHPriceOracle` — (`contracts/LRTOracle.sol`, `contracts/oracles/RETHPriceOracle.sol`)

---

### Summary

`LRTOracle.updateRSETHPrice()` is callable by any address. When rETH's exchange rate drops enough to push `newRsETHPrice` more than `pricePercentageLimit` below `highestRsethPrice`, the downside-protection branch in `_updateRsETHPrice()` pauses `LRTDepositPool`, `LRTWithdrawalManager`, and `LRTOracle` itself. Because `RETHPriceOracle` has no staleness or bounds guard, any transient or real rate dip in Rocket Pool's `getExchangeRate()` is sufficient to arm this path. No privileged role is required.

---

### Finding Description

**Entry point — no access control:**

`updateRSETHPrice()` carries only the `whenNotPaused` modifier; there is no role check. [1](#0-0) 

**Oracle — no staleness or bounds guard:**

`RETHPriceOracle.getAssetPrice()` blindly forwards the live Rocket Pool exchange rate with zero validation. [2](#0-1) 

**Auto-pause branch — triggered by any caller:**

Inside `_updateRsETHPrice()`, if the computed `newRsETHPrice` falls more than `pricePercentageLimit` below `highestRsethPrice`, the function pauses all three contracts and returns early — with no check on `msg.sender`. [3](#0-2) 

The `_pause()` helper itself is unconditional: [4](#0-3) 

**Precondition is realistic:**

`pricePercentageLimit` is set by admin (e.g., 1 % = `1e16`). A Rocket Pool slashing event, a temporary liquidity imbalance, or even normal rETH rate oscillation can produce a 1–2 % dip in `getExchangeRate()`. Because `highestRsethPrice` is the all-time high, any sustained dip relative to that peak satisfies the condition. [5](#0-4) 

---

### Impact Explanation

Once the branch fires, `LRTDepositPool.pause()`, `LRTWithdrawalManager.pause()`, and `LRTOracle._pause()` are all called. All deposits and withdrawals are frozen until an admin manually unpauses each contract. This is **temporary freezing of funds** — a valid Medium impact per the scope rules.

---

### Likelihood Explanation

- `updateRSETHPrice()` is public; any EOA or contract can call it at any time.
- `RETHPriceOracle` has no circuit breaker, so any live rate dip is immediately reflected.
- `pricePercentageLimit` is expected to be set to a small value (1–5 %) to protect against large drops, making the threshold easy to cross during normal market volatility.
- No capital, no special role, and no front-running is required — a single transaction suffices.

---

### Recommendation

1. **Add a caller restriction to `updateRSETHPrice()`** — gate it with `onlyLRTManager` or a dedicated `PRICE_UPDATER_ROLE`, similar to `updateRSETHPriceAsManager()`.
2. **Add staleness and bounds validation to `RETHPriceOracle`** — reject prices that deviate more than a configurable band from the last accepted value, or that are older than a maximum age.
3. **Separate the auto-pause trigger from the public price-update path** — the downside-protection pause should only be reachable by privileged callers or a dedicated keeper with a time-lock.

---

### Proof of Concept

```solidity
// SPDX-License-Identifier: MIT
pragma solidity 0.8.27;

// Foundry fork test (mainnet fork)
import "forge-std/Test.sol";

contract ExploitTest is Test {
    ILRTOracle oracle = ILRTOracle(<deployed_oracle>);
    ILRTDepositPool pool = ILRTDepositPool(<deployed_pool>);
    ILRTWithdrawalManager wm = ILRTWithdrawalManager(<deployed_wm>);
    IrETH reth = IrETH(<reth_address>);

    function testUnprivilegedPause() external {
        // 1. Admin sets pricePercentageLimit to 1% (1e16)
        vm.prank(admin);
        oracle.setPricePercentageLimit(1e16);

        // 2. Record current highestRsethPrice (already set from prior updates)
        uint256 peak = oracle.highestRsethPrice();

        // 3. Mock rETH.getExchangeRate() to return a value 2% below the
        //    value that would produce the current peak rsETH price
        uint256 dippedRate = reth.getExchangeRate() * 98 / 100;
        vm.mockCall(
            address(reth),
            abi.encodeWithSelector(IrETH.getExchangeRate.selector),
            abi.encode(dippedRate)
        );

        // 4. Unprivileged caller triggers the auto-pause
        vm.prank(address(1));
        oracle.updateRSETHPrice();

        // 5. Assert all three contracts are paused
        assertTrue(pool.paused(),   "DepositPool not paused");
        assertTrue(wm.paused(),     "WithdrawalManager not paused");
        assertTrue(oracle.paused(), "LRTOracle not paused");
    }
}
```

The test requires no privileged role. It passes on unmodified production code whenever the mocked rate causes `newRsETHPrice` to fall more than `pricePercentageLimit` below `highestRsethPrice`. [1](#0-0) [2](#0-1) [3](#0-2)
