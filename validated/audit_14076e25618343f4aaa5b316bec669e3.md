### Title
Unelected Committee Member Can Pre-Cast Governance Votes via Block-Producer MEMPOOL Bypass at Protocol Version 10 — (File: `eras/conway/impl/src/Cardano/Ledger/Conway/Rules/GovCert.hs`)

---

### Summary

The `GOVCERT` rule allows a cold credential to authorize a hot key as soon as it appears in any pending `UpdateCommittee` proposal, before that proposal is ratified. At protocol version 10, the `GOV` rule contains no check for unelected committee member votes; the guard exists only in the `MEMPOOL` rule, which block producers can bypass by including transactions directly in blocks. The `GOV` rule then stores the pre-cast vote in `gasCommitteeVotes`, and once the `UpdateCommittee` proposal is enacted the vote is counted toward ratification of other governance actions.

---

### Finding Description

**Root cause 1 — premature hot-key authorization (`GovCert.hs`)**

`checkAndOverwriteCommitteeMemberState` (lines 190–208) permits a cold credential to authorize a hot key if it is either a current committee member (`isCurrentMember`) **or** appears in any pending `UpdateCommittee` proposal (`isPotentialFutureMember`):

```haskell
let isCurrentMember =
      strictMaybe False (Map.member coldCred . committeeMembers) cgceCurrentCommittee
    committeeUpdateContainsColdCred GovActionState {gasProposalProcedure} =
      case pProcGovAction gasProposalProcedure of
        UpdateCommittee _ _ newMembers _ -> Map.member coldCred newMembers
        _ -> False
    isPotentialFutureMember =
      any committeeUpdateContainsColdCred cgceCommitteeProposals
isCurrentMember || isPotentialFutureMember
  ?! (injectFailure . ConwayCommitteeIsUnknown) coldCred
``` [1](#0-0) 

The `cgceCommitteeProposals` field carries **all** live `UpdateCommittee` proposals, regardless of ratification status. [2](#0-1) 

**Root cause 2 — unelected-voter guard absent from `GOV` rule at PV 10**

The `GOV` rule only applies `unelectedCommitteeVoters` when `hardforkConwayDisallowUnelectedCommitteeFromVoting` is true, which requires protocol version ≥ 11:

```haskell
when (hardforkConwayDisallowUnelectedCommitteeFromVoting $ pp ^. ppProtocolVersionL) $
  failOnNonEmpty
    (unelectedCommitteeVoters committee committeeState gsVotingProcedures)
    (injectFailure . UnelectedCommitteeVoters)
``` [3](#0-2) 

```haskell
hardforkConwayDisallowUnelectedCommitteeFromVoting :: ProtVer -> Bool
hardforkConwayDisallowUnelectedCommitteeFromVoting pv = pvMajor pv > natVersion @10
``` [4](#0-3)
