Audit Report

## Title
Protocol Fee Charged on Pause-Period Yield Due to Stale Price Baseline — (`contracts/LRTOracle.sol`)

## Summary

`LRTOracle._updateRsETHPrice()` includes a `protocolPaused` guard intended to suppress fee minting while the protocol is paused. However, because `updateRSETHPrice()` is blocked by `whenNotPaused` during a pause, `rsETHPrice` is never updated during the pause period. On the first call after unpause, `protocolPaused` evaluates to `false`, and the fee is computed against the stale pre-pause price baseline — causing the full yield that accrued during the pause to be subject to the protocol fee, contrary to the design intent.

## Finding Description

`updateRSETHPrice()` is gated by `whenNotPaused`:

```solidity
// LRTOracle.sol L87-89
function updateRSETHPrice() public whenNotPaused {
    _updateRsETHPrice();
}
```

Inside `_updateRsETHPrice()`, `previousTVL` is derived from the stored `rsETHPrice`:

```solidity
// LRTOracle.sol L234
uint256 previousTVL = rsethSupply.mulWad(rsETHPrice);
```

The pause guard is evaluated at call time:

```solidity
// LRTOracle.sol L240-246
bool protocolPaused = lrtDepositPool.paused() || withdrawalManager.paused() || paused;

uint256 protocolFeeInETH = 0;
if (!protocolPaused && totalETHInProtocol > previousTVL) {
    uint256 rewardAmount = totalETHInProtocol - previousTVL;
    protocolFeeInETH = (rewardAmount * lrtConfig.protocolFeeInBPS()) / 10_000;
}
```

`pauseAll()` in `LRTConfig` pauses the oracle along with all other contracts:

```solidity
// LRTConfig.sol L262-270
function pauseAll() external onlyRole(LRTConstants.PAUSER_ROLE) {
    ...
    if (!lrtOracle.paused()) lrtOracle.pause();
    ...
}
```

After unpause, `protocolPaused` is `false`, but `rsETHPrice` still holds the pre-pause value. The entire TVL increase from the pause period is treated as `rewardAmount`, and the full protocol fee is charged on it. The `updateRSETHPriceAsManager()` bypass exists but is not enforced during pauses and requires privileged action that is not part of the standard unpause flow.

## Impact Explanation

rsETH holders lose a portion of yield that accrued during the pause period to protocol fees. The fee is minted as new rsETH to the treasury, diluting all existing holders. Users could not withdraw or rebalance during the pause, yet they bear the full fee on pause-period yield. This constitutes **theft of unclaimed yield** (High severity), which is an explicitly allowed impact in scope.

## Likelihood Explanation

Every standard `pauseAll()` invocation followed by unpause — without a manual `updateRSETHPriceAsManager()` call during the pause — triggers this automatically. No attacker action is required beyond calling the public `updateRSETHPrice()` after unpause. Staking rewards accrue continuously, so any non-trivial pause duration produces measurable impact. Likelihood is **Medium**: pauses are infrequent but the impact is automatic and requires no special capability.

## Recommendation

On unpause, advance the `rsETHPrice` baseline to the current TVL-derived price without charging a fee, so only yield accruing after the unpause is fee-eligible. One concrete approach: call `_updateRsETHPrice()` with fee suppressed (e.g., via a flag or a dedicated internal function) as part of the `unpause()` flow in `LRTOracle`, before any public fee-eligible call can occur. Alternatively, record the block/timestamp of the most recent unpause and skip the fee in `_updateRsETHPrice()` if `previousTVL` was set before that timestamp.

## Proof of Concept

1. Protocol is running; `updateRSETHPrice()` is called, setting `rsETHPrice = P₀`.
2. Admin calls `pauseAll()` — `LRTDepositPool`, `LRTWithdrawalManager`, and `LRTOracle` are all paused (`LRTConfig.sol` L268–270).
3. 30 days pass. EigenLayer staking rewards accrue; `_getTotalEthInProtocol()` would now return `TVL₁ > TVL₀`. `rsETHPrice` remains `P₀` (no update possible due to `whenNotPaused`).
4. Admin calls `unpause()` on all contracts.
5. Any user calls `updateRSETHPrice()`:
   - `previousTVL = rsethSupply * P₀` (stale, pre-pause value)
   - `totalETHInProtocol = TVL₁` (includes 30 days of rewards)
   - `protocolPaused = false` (all contracts unpaused)
   - `rewardAmount = TVL₁ - TVL₀` (entire pause-period yield)
   - `protocolFeeInETH = rewardAmount * protocolFeeInBPS / 10_000`
   - rsETH minted to treasury for the full fee on pause-period yield (`LRTOracle.sol` L299–308)
