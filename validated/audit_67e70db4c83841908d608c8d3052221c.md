### Title
Unvalidated `ProposalInit.builder` Allows Proposer to Redirect All Block Fees to an Arbitrary Address — (`File: crates/apollo_consensus_orchestrator/src/validate_proposal.rs`)

---

### Summary

The `builder` field of `ProposalInit` is a proposer-supplied address that is used verbatim as the `sequencer_address` in the `BlockInfo` passed to the blockifier for execution. It is also chained into the `PartialBlockHashComponents` (and thus the `ProposalCommitment`). However, `is_proposal_init_valid` — the sole gate that validators use to accept or reject a proposal's metadata — never checks `builder`. A malicious proposer can set `builder` to any address they control, redirect all block fees to that address, and cause every validator to accept and commit the block.

---

### Finding Description

`ProposalInit.builder` is defined as "Address of the one who builds/sequences the block." [1](#0-0) 

`convert_to_sn_api_block_info` maps `init.builder` directly to `sequencer_address` in the `BlockInfo` that is handed to the blockifier for both proposal building and validation: [2](#0-1) 

The same `sequencer_address` is then captured in `PartialBlockHashComponents` and chained into the block hash / `ProposalCommitment`: [3](#0-2) [4](#0-3) 

`is_proposal_init_valid` — the complete validator-side check — validates `timestamp`, `starknet_version`, `version_constant_commitment`, `height`, `l1_da_mode`, `l2_gas_price_fri`, all four L1 gas prices (within margin), and `fee_proposal_fri`. It does **not** check `builder` at all: [5](#0-4) 

`ProposalInitValidation` — the struct that carries the validator's reference values — has no `builder` field: [6](#0-5) 

The earlier `handle_proposal` check in `single_height_consensus.rs` only validates `height` and `proposer`; it does not validate `builder` either: [7](#0-6) 

After consensus commits the block, `finalize_decision` calls `convert_to_sn_api_block_info(init)` again and passes the result to `update_state_sync_with_new_block`, which writes `SequencerContractAddress(init.builder)` into the persisted `BlockHeaderWithoutHash`: [8](#0-7) [9](#0-8) 

---

### Impact Explanation

**Critical — Incorrect fee/balance with economic impact; wrong state committed.**

The `sequencer_address` in `BlockInfo` is the address to which the blockifier credits all transaction fees for the block. By setting `builder` to an attacker-controlled address, the proposer:

1. **Steals all block fees.** Every fee-paying transaction in the block pays fees to `attacker_address` instead of the legitimate sequencer.
2. **Corrupts the committed block header.** The persisted `BlockHeaderWithoutHash.sequencer` and the on-chain block hash both record `attacker_address` as the sequencer. This is an incorrect state commitment.
3. **Corrupts `get_sequencer_address` syscall results.** Any contract that calls `get_sequencer_address` during execution of that block receives `attacker_address`, potentially affecting contract logic that depends on the sequencer identity.

Because `builder` is included in `PartialBlockHashComponents.sequencer` and thus in the `ProposalCommitment`, the proposer's and all validators' commitments agree — consensus reaches a valid quorum on the corrupted block.

---

### Likelihood Explanation

**High.** Any validator whose turn it is to propose can exploit this in a single block. No special precondition is required beyond being the scheduled proposer. The attack is silent: no error is logged, no metric is incremented, and the block is indistinguishable from a legitimate one to all other validators.

---

### Recommendation

Add `builder` (expected sequencer address) to `ProposalInitValidation` and enforce it in `is_proposal_init_valid`. The expected value should be derived from the validator's own configuration (its `builder_address`) or from a registered mapping of proposer → builder address. Concretely:

1. Add `expected_builder: ContractAddress` to `ProposalInitValidation`.
2. In `is_proposal_init_valid`, reject any `init` where `init.builder != proposal_init_validation.expected_builder`.
3. Populate `expected_builder` in `validate_proposal` from the node's own `builder_address` config (the same value used in `ProposalBuildArguments.builder_address`).

---

    init_proposed: &ProposalInit,
    clock: &dyn Clock,
    l1_gas_price_provider: Arc<dyn L1GasPriceProviderClient>,
    gas_price_params: &GasPriceParams,

    let now: u64 = clock.unix_now();

        proposal_init_validation.previous_proposal_init.as_ref().map_or(0, |info| info.timestamp);
    if init_proposed.timestamp < last_block_timestamp {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "Timestamp is too old: last_block_timestamp={}, proposed={}",
                last_block_timestamp, init_proposed.timestamp
            ),
        ));
    }
    if init_proposed.timestamp > now + proposal_init_validation.block_timestamp_window_seconds {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "Timestamp is in the future: now={}, block_timestamp_window_seconds={}, \
                 proposed={}",
                now,
                proposal_init_validation.block_timestamp_window_seconds,
                init_proposed.timestamp
            ),
        ));
    }
    if init_proposed.starknet_version != proposal_init_validation.starknet_version {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "starknet_version mismatch: expected={:?}, proposed={:?}",
                proposal_init_validation.starknet_version, init_proposed.starknet_version
            ),
        ));

    // sentinel, so reject anything else. Enforcing the same value the proposer emits keeps the two

                init_proposed.version_constant_commitment
            ),
        ));
    }

        && init_proposed.l2_gas_price_fri == proposal_init_validation.l2_gas_price_fri)
    {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),

        ));
    }
    let (l1_gas_prices_fri, l1_gas_prices_wei) = get_l1_prices_in_fri_and_wei(

        init_proposed.timestamp,
        proposal_init_validation.previous_proposal_init.as_ref(),
        gas_price_params,
    )
    .await;
    let l1_gas_price_margin_percent =
        VersionedConstants::latest_constants().l1_gas_price_margin_percent.into();
    debug!("L1 price info: fri={l1_gas_prices_fri:?}, wei={l1_gas_prices_wei:?}");

    let l1_gas_price_fri = l1_gas_prices_fri.l1_gas_price;
    let l1_data_gas_price_fri = l1_gas_prices_fri.l1_data_gas_price;
    let l1_gas_price_wei = l1_gas_prices_wei.l1_gas_price;
    let l1_data_gas_price_wei = l1_gas_prices_wei.l1_data_gas_price;
    let l1_gas_price_fri_proposed = init_proposed.l1_gas_price_fri;
    let l1_data_gas_price_fri_proposed = init_proposed.l1_data_gas_price_fri;
    let l1_gas_price_wei_proposed = init_proposed.l1_gas_price_wei;
    let l1_data_gas_price_wei_proposed = init_proposed.l1_data_gas_price_wei;

            l1_data_gas_price_fri,
            l1_gas_price_margin_percent,

        && within_margin(l1_gas_price_wei_proposed, l1_gas_price_wei, l1_gas_price_margin_percent)
        && within_margin(
            l1_data_gas_price_wei_proposed,
            l1_data_gas_price_wei,
            l1_gas_price_margin_percent,

        return Err(ValidateProposalError::InvalidProposalInit(

            proposal_init_validation.clone(),

                 FRI={l1_data_gas_price_fri}, proposed={l1_data_gas_price_fri_proposed}, expected \

                init_proposed.clone(),
                proposal_init_validation.clone(),
                format!(
                    "fee_proposal must be absent before V0_14_3, got Some at version {}",
                    init_proposed.starknet_version
                ),
            ));
        }
        (None, true) => {
            return Err(ValidateProposalError::InvalidProposalInit(
                init_proposed.clone(),
                proposal_init_validation.clone(),
                format!(
                    "fee_proposal is required at V0_14_3+, got None at version {}",
                    init_proposed.starknet_version
                ),
            ));
        }
        _ => {}
    }

    // Validate fee_proposal is within the configured margin of fee_actual.
    // During initiation (fee_actual is None, <window_size blocks), bounds are not enforced.
    if let (Some(fee_actual), Some(fee_proposal)) =
        (proposal_init_validation.fee_actual, init_proposed.fee_proposal_fri)
    {
        let (lower_bound, upper_bound) = fee_proposal_bounds(
            fee_actual,
            VersionedConstants::latest_constants().fee_proposal_margin_ppt,

            return Err(ValidateProposalError::InvalidProposalInit(
                init_proposed.clone(),
                proposal_init_validation.clone(),
                format!(
                    "Fee proposal out of bounds: fee_actual={}, fee_proposal={}, allowed \
                     range=[{lower_bound}, {upper_bound}]",
                    fee_actual.0, fee_proposal.0
                ),
            ));
        }
    }

    Ok(())
