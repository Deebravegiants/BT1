### Title
Proposer-Controlled `builder` Field in `ProposalInit` Is Never Validated by Validators, Enabling Arbitrary `sequencer_address` Injection into Block Execution - (File: `crates/apollo_consensus_orchestrator/src/utils.rs`)

### Summary

The `builder` field of `ProposalInit` is set freely by the proposer from its static configuration and is transmitted over the wire to all validators. `convert_to_sn_api_block_info` maps `init.builder` directly to `BlockInfo.sequencer_address`, which is then used for fee collection and the `get_sequencer_address` syscall. Validators never check the `builder` field in `is_proposal_init_valid`, so any proposer can inject an arbitrary address as the block's sequencer address, redirecting all transaction fees and corrupting the `get_sequencer_address` syscall result for every transaction in the block.

### Finding Description

`ProposalInit` carries a `builder` field defined as "Address of the one who builds/sequences the block." [1](#0-0) 

During block building, the proposer sets this field from its own static configuration: [2](#0-1) 

`convert_to_sn_api_block_info` then maps `init.builder` directly to `sequencer_address` in the `BlockInfo` struct that is passed to the batcher/blockifier for both the proposer and validator paths: [3](#0-2) 

The `sequencer_address` is then included in the partial block hash computation: [4](#0-3) 

and chained into the final block hash: [5](#0-4) 

The validator's `is_proposal_init_valid` function checks `height`, `l1_da_mode`, `l2_gas_price_fri`, `starknet_version`, `version_constant_commitment`, L1 gas prices, `fee_proposal_fri`, and `timestamp` — but **never checks `builder`**: [6](#0-5) 

The `ProposalInitValidation` struct that carries the validator's reference values for these checks has no field for `builder`: [7](#0-6) 

### Impact Explanation

**Fee redirection (economic impact):** In Starknet, transaction fees are transferred to `BlockInfo.sequencer_address`. Because `sequencer_address` is set from the unvalidated `builder` field, a malicious proposer can set `builder` to any address they control and receive all transaction fees from every block they propose, regardless of what the legitimate sequencer address should be.

**Wrong `get_sequencer_address` syscall result:** The Cairo OS exposes `sequencer_address` from `BlockInfo` via the `get_sequencer_address` syscall. Any contract that calls this syscall during a block proposed by a malicious node will receive the injected address, not the legitimate sequencer address. This corrupts execution results, receipts, and events for any contract that branches on the sequencer address (e.g., access-control checks, fee-routing logic). [8](#0-7) 

**Wrong block hash committed to consensus:** Because `sequencer_address` is chained into the block hash, the committed `PartialBlockHash` and the final `BlockHash` both reflect the injected address. Validators accept and vote on this hash without knowing the `builder` value was arbitrary.

### Likelihood Explanation

Any node that is legitimately selected as a proposer for a round can exploit this. The proposer simply sets `builder_address` in its static configuration to any address before the round begins. No special privilege beyond being a scheduled proposer is required. Validators have no mechanism to detect or reject the manipulated field because it is absent from `ProposalInitValidation`. The attack is silent — the proposal passes all existing checks and produces a valid `ProposalCommitment`. [9](#0-8) 

### Recommendation

Add `builder` to `ProposalInitValidation` and enforce it in `is_proposal_init_valid`. The expected value should be derived from the committee's authoritative record of the proposer's builder address (analogous to how `proposer` is checked against `committee.get_proposer(height, round)` in `handle_proposal`). Until the committee carries a per-member builder address, the validator should at minimum reject any `builder` value that is the zero address or that differs from the proposer's own registered address. [10](#0-9) 

### Proof of Concept

1. A node is legitimately selected as proposer for height H, round R.
2. Before the round, the operator sets `builder_address = 0xATTACKER` in the node's static config.
3. `build_proposal` → `initiate_build` constructs `ProposalInit { builder: 0xATTACKER, ... }` and broadcasts it.
4. Each validator calls `validate_proposal` → `is_proposal_init_valid`. The function checks height, gas prices, starknet version, etc., but has no check on `builder`. Validation passes.
5. `initiate_validation` calls `convert_to_sn_api_block_info(&init)`, producing `BlockInfo { sequencer_address: 0xATTACKER, ... }`.
6. The batcher executes all transactions with `sequencer_address = 0xATTACKER`. Every fee transfer goes to `0xATTACKER`. Every `get_sequencer_address` syscall returns `0xATTACKER`.
7. `PartialBlockHashComponents::new` chains `0xATTACKER` into the block hash. Validators vote `Precommit` on this hash. The block is finalized with the injected sequencer address embedded in the canonical block hash. [11](#0-10) [12](#0-11)

pub(crate) fn convert_to_sn_api_block_info(
    init: &ProposalInit,
) -> Result<starknet_api::block::BlockInfo, StarknetApiError> {
    if init.l1_gas_price_fri.0 == 0
        || init.l1_gas_price_wei.0 == 0
        || init.l1_data_gas_price_fri.0 == 0
        || init.l1_data_gas_price_wei.0 == 0
        || init.l2_gas_price_fri.0 == 0
    {
        warn!("Zero gas price detected in block info: {:?}", init);
    }

    let l1_data_gas_price_fri = NonzeroGasPrice::new(init.l1_data_gas_price_fri)?;
    let l1_gas_price_wei = NonzeroGasPrice::new(init.l1_gas_price_wei)?;
    let l1_data_gas_price_wei = NonzeroGasPrice::new(init.l1_data_gas_price_wei)?;
    let l2_gas_price_fri = NonzeroGasPrice::new(init.l2_gas_price_fri)?;
    let proposal_init_info = PreviousProposalInitInfo::from(init);
    let eth_to_fri_rate = calculate_eth_to_fri_rate(&proposal_init_info)?;

    let l2_gas_price_wei = NonzeroGasPrice::new(init.l2_gas_price_fri.fri_to_wei(eth_to_fri_rate)?)

                init.l2_gas_price_fri
            )
        })?;
    Ok(starknet_api::block::BlockInfo {
        block_number: init.height,
        block_timestamp: BlockTimestamp(init.timestamp),
        sequencer_address: init.builder,
        gas_prices: GasPrices {
            strk_gas_prices: GasPriceVector {

                l1_data_gas_price: l1_data_gas_price_fri,

                l1_gas_price: l1_gas_price_wei,
                l1_data_gas_price: l1_data_gas_price_wei,
                l2_gas_price: l2_gas_price_wei,
            },
        },
        use_kzg_da: init.l1_da_mode.is_use_kzg_da(),
        starknet_version: init.starknet_version,
    })

#[instrument(level = "warn", skip_all, fields(?proposal_init_validation, ?init_proposed))]
async fn is_proposal_init_valid(
    proposal_init_validation: &ProposalInitValidation,
    init_proposed: &ProposalInit,
    clock: &dyn Clock,
    l1_gas_price_provider: Arc<dyn L1GasPriceProviderClient>,
    gas_price_params: &GasPriceParams,
) -> ValidateProposalResult<()> {
    let now: u64 = clock.unix_now();
    let last_block_timestamp =
        proposal_init_validation.previous_proposal_init.as_ref().map_or(0, |info| info.timestamp);
    if init_proposed.timestamp < last_block_timestamp {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "Timestamp is too old: last_block_timestamp={}, proposed={}",
                last_block_timestamp, init_proposed.timestamp
            ),
        ));
    }
    if init_proposed.timestamp > now + proposal_init_validation.block_timestamp_window_seconds {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "Timestamp is in the future: now={}, block_timestamp_window_seconds={}, \
                 proposed={}",
                now,
                proposal_init_validation.block_timestamp_window_seconds,
                init_proposed.timestamp
            ),
        ));
    }
    if init_proposed.starknet_version != proposal_init_validation.starknet_version {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "starknet_version mismatch: expected={:?}, proposed={:?}",
                proposal_init_validation.starknet_version, init_proposed.starknet_version
            ),
        ));
    }
    // `version_constant_commitment` is proposer-supplied (network-derived). It is not yet a real
    // commitment (see `expected_version_constant_commitment`): the only valid value is the
    // sentinel, so reject anything else. Enforcing the same value the proposer emits keeps the two
    // sides in lockstep, so a real value cannot ship on one side without the other.
    let expected_commitment = expected_version_constant_commitment();
    if init_proposed.version_constant_commitment != expected_commitment {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "version_constant_commitment mismatch: expected={expected_commitment:?}, \
                 proposed={:?}",
                init_proposed.version_constant_commitment
            ),
        ));
    }
    if !(init_proposed.height == proposal_init_validation.height
        && init_proposed.l1_da_mode == proposal_init_validation.l1_da_mode
        && init_proposed.l2_gas_price_fri == proposal_init_validation.l2_gas_price_fri)
    {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            "ProposalInit validation failed".to_string(),
        ));
    }
    let (l1_gas_prices_fri, l1_gas_prices_wei) = get_l1_prices_in_fri_and_wei(
        l1_gas_price_provider,
        init_proposed.timestamp,
        proposal_init_validation.previous_proposal_init.as_ref(),
        gas_price_params,
    )
    .await;
    let l1_gas_price_margin_percent =
        VersionedConstants::latest_constants().l1_gas_price_margin_percent.into();
    debug!("L1 price info: fri={l1_gas_prices_fri:?}, wei={l1_gas_prices_wei:?}");

    let l1_gas_price_fri = l1_gas_prices_fri.l1_gas_price;
    let l1_data_gas_price_fri = l1_gas_prices_fri.l1_data_gas_price;
    let l1_gas_price_wei = l1_gas_prices_wei.l1_gas_price;
    let l1_data_gas_price_wei = l1_gas_prices_wei.l1_data_gas_price;
    let l1_gas_price_fri_proposed = init_proposed.l1_gas_price_fri;
    let l1_data_gas_price_fri_proposed = init_proposed.l1_data_gas_price_fri;
    let l1_gas_price_wei_proposed = init_proposed.l1_gas_price_wei;
    let l1_data_gas_price_wei_proposed = init_proposed.l1_data_gas_price_wei;

    if !(within_margin(l1_gas_price_fri_proposed, l1_gas_price_fri, l1_gas_price_margin_percent)
        && within_margin(
            l1_data_gas_price_fri_proposed,
            l1_data_gas_price_fri,
            l1_gas_price_margin_percent,
        )
        && within_margin(l1_gas_price_wei_proposed, l1_gas_price_wei, l1_gas_price_margin_percent)
        && within_margin(
            l1_data_gas_price_wei_proposed,
            l1_data_gas_price_wei,
            l1_gas_price_margin_percent,
        ))
    {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "L1 gas price mismatch: expected L1 gas price FRI={l1_gas_price_fri}, \
                 proposed={l1_gas_price_fri_proposed}, expected L1 data gas price \
                 FRI={l1_data_gas_price_fri}, proposed={l1_data_gas_price_fri_proposed}, expected \
                 L1 gas price WEI={l1_gas_price_wei}, proposed={l1_gas_price_wei_proposed}, \
                 expected L1 data gas price WEI={l1_data_gas_price_wei}, \
                 proposed={l1_data_gas_price_wei_proposed}, \
                 l1_gas_price_margin_percent={l1_gas_price_margin_percent}"
            ),
        ));
    }

    // fee_proposal is required iff Starknet version >= V0_14_3.
    let fee_proposal_required = init_proposed.starknet_version >= StarknetVersion::V0_14_3;
    match (init_proposed.fee_proposal_fri, fee_proposal_required) {
        (Some(_), false) => {
            return Err(ValidateProposalError::InvalidProposalInit(
                init_proposed.clone(),
                proposal_init_validation.clone(),
                format!(
                    "fee_proposal must be absent before V0_14_3, got Some at version {}",
                    init_proposed.starknet_version
                ),
            ));
        }
        (None, true) => {
            return Err(ValidateProposalError::InvalidProposalInit(
                init_proposed.clone(),
                proposal_init_validation.clone(),
                format!(
                    "fee_proposal is required at V0_14_3+, got None at version {}",
                    init_proposed.starknet_version
                ),
            ));
        }
        _ => {}
    }

    // Validate fee_proposal is within the configured margin of fee_actual.
    // During initiation (fee_actual is None, <window_size blocks), bounds are not enforced.
    if let (Some(fee_actual), Some(fee_proposal)) =
        (proposal_init_validation.fee_actual, init_proposed.fee_proposal_fri)
    {
        let (lower_bound, upper_bound) = fee_proposal_bounds(
            fee_actual,
            VersionedConstants::latest_constants().fee_proposal_margin_ppt,
        );
        if fee_proposal.0 < lower_bound || fee_proposal.0 > upper_bound {
            return Err(ValidateProposalError::InvalidProposalInit(
                init_proposed.clone(),
                proposal_init_validation.clone(),
                format!(
                    "Fee proposal out of bounds: fee_actual={}, fee_proposal={}, allowed \
                     range=[{lower_bound}, {upper_bound}]",
                    fee_actual.0, fee_proposal.0
                ),
            ));
        }
    }

    Ok(())
}
