### Title
Unauthenticated Gossipsub Vote Forgery Bypasses Consensus Quorum — (`crates/apollo_consensus/src/single_height_consensus.rs`)

---

### Summary

Any peer connected to the gossipsub network can broadcast `Vote` messages claiming to be from any committee validator, with an empty or arbitrary signature. Because `handle_vote` in `SingleHeightConsensus` contains only a `// TODO(Asmaa): verify the signature` comment and performs no actual cryptographic check, forged votes are counted toward the 2/3+1 Byzantine quorum threshold. A single malicious peer can forge enough precommit votes to push a node past the decision threshold, causing it to commit a block whose validator-set attestation is entirely fabricated.

---

### Finding Description

**Deserialization** (`crates/apollo_protobuf/src/converters/consensus.rs:86-88`): The protobuf `TryFrom<protobuf::Vote> for Vote` conversion explicitly defaults the signature to `RawSignature::default()` (all-zero bytes) when the wire field is absent:

```rust
// Convert Hashes to RawSignature (default to empty if None)
let signature =
    value.signature.map(|hashes| hashes.try_into()).transpose()?.unwrap_or_default();
``` [1](#0-0) 

**Manager layer** (`crates/apollo_consensus/src/manager.rs:993-996`): After height-routing, the manager calls `shc.handle_vote(message)` with no signature check: [2](#0-1) 

**SHC layer** (`crates/apollo_consensus/src/single_height_consensus.rs:241-252`): `handle_vote` acknowledges the missing check with a TODO comment, then only validates height and committee membership by address — never the cryptographic signature:

```rust

    if !self.committee.members().iter().any(|s| s.address == vote.voter) {

**Quorum accumulation** (`crates/apollo_consensus/src/state_machine.rs:194-203`): Once a vote passes the address check, `vote_weight` looks up the staker's weight by address and adds it to the running total: [4](#0-3) 

**Verification function exists but is never called** (`crates/apollo_signature_manager/src/signature_manager.rs:179-186`): `verify_precommit_vote_signature` is implemented and tested in isolation, but is not invoked anywhere in the gossipsub vote processing path: [5](#0-4) 

The `Staker` struct carries a `public_key` field that is available for verification but is never consulted during vote processing: [6](#0-5) 

Self-votes produced by the node itself also use `RawSignature::default()` (another TODO), confirming the entire vote-signing/verification subsystem is unimplemented end-to-end: [7](#0-6) 

---

### Impact Explanation

An attacker who can send gossipsub messages (any peer on the p2p network) can:

1. Read the current committee from the public staking contract to obtain all validator `ContractAddress` values.
2. Craft `Vote` protobuf messages with `vote_type=Precommit`, the current `height`/`round`, the target `proposal_commitment`, `voter` set to each committee member's address, and `signature` omitted (defaults to empty).
3. Broadcast these messages over gossipsub.
4. The receiving node counts each forged vote at full staking weight. Once the accumulated weight exceeds the 2/3 Byzantine quorum threshold (`VotesThreshold::is_met`), `DecisionReached` fires and the node commits the block.

The committed block's validator-set attestation records validators as having signed who never did. If the rest of the network did not legitimately reach quorum, the node is on a divergent fork with corrupted state. This is a **Critical consensus integrity** impact: the node commits a block under false quorum, applying all state transitions (balances, storage, events, receipts) from that block to its local state.

---

### Likelihood Explanation

- **Attacker entry point**: unauthenticated gossipsub peer — no special privileges required.
- **Required knowledge**: committee member addresses, which are public on-chain.
- **Required capability**: ability to send gossipsub messages, which is the normal operation of any network peer.
- **Blocking controls**: none — the duplicate/conflict check only prevents the same `(round, voter)` pair from being counted twice, which the attacker trivially avoids by using distinct validator addresses.

---

### Recommendation

Call `verify_precommit_vote_signature` (or `verify_identity` for prevotes) inside `SingleHeightConsensus::handle_vote` before accepting any peer vote. The `Staker::public_key` field is already present in the committee data. Reject and report the peer on verification failure. Resolve both TODOs (`// TODO(Asmaa): verify the signature` and `// TODO(Asmaa): sign the vote`) together so that self-votes are also signed before broadcast.

---

### Proof of Concept

```rust
// In single_height_consensus_test.rs — demonstrates forged precommit with empty signature
// reaches quorum and triggers DecisionReached without any real validator key.
#[test]
fn forged_precommit_with_empty_signature_reaches_quorum() {
    use starknet_api::crypto::utils::RawSignature;

    let mut shc = new_shc(*VALIDATOR_ID_1, false);
    let init = proposal_init(HEIGHT_0, ROUND_0, *PROPOSER_ID);
    let round = init.round;
    shc.handle_proposal(init);
    shc.handle_event(StateMachineEvent::FinishedValidation(Some(BLOCK.id), round, None));

    // Forge prevotes with RawSignature::default() for two committee members.
    let mut forged_prevote = |voter| Vote {
        vote_type: VoteType::Prevote,
        height: HEIGHT_0,
        round: ROUND_0,
        proposal_commitment: Some(BLOCK.id),
        voter,
        signature: RawSignature::default(), // empty — never verified
    };
    let _ = shc.handle_vote(forged_prevote(*PROPOSER_ID));
    let _ = shc.handle_vote(forged_prevote(*VALIDATOR_ID_2));
    // prevote quorum reached with forged votes

    // Forge precommits with RawSignature::default().
    let mut forged_precommit = |voter| Vote {
        vote_type: VoteType::Precommit,
        height: HEIGHT_0,
        round: ROUND_0,
        proposal_commitment: Some(BLOCK.id),
        voter,
        signature: RawSignature::default(),
    };
    let _ = shc.handle_vote(forged_precommit(*PROPOSER_ID));
    let mut decision = shc.handle_vote(forged_precommit(*VALIDATOR_ID_2));

    // DecisionReached fires — block committed on forged votes, no real signatures checked.
    assert_decision(&mut decision, ROUND_0, BLOCK.id);
}
```
