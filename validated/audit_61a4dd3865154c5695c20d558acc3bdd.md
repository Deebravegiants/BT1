### Title
Proposer-Supplied `builder` Address in `ProposalInit` Accepted Without Validator-Side Binding, Enabling Fee Redirection to Arbitrary Address - (File: `crates/apollo_consensus_orchestrator/src/validate_proposal.rs`)

### Summary

`is_proposal_init_valid` validates many `ProposalInit` fields against locally-derived values (height, `l1_da_mode`, `l2_gas_price_fri`, `starknet_version`, `version_constant_commitment`, L1 gas prices, `fee_proposal`), but never checks the `builder` field. The `builder` field is then passed verbatim as `sequencer_address` into `BlockInfo` for execution. A malicious proposer can set `builder` to any address, redirecting all block fees to an attacker-controlled address while the proposal passes all validation checks.

### Finding Description

`ProposalInit.builder` is described as "Address of the one who builds/sequences the block." [1](#0-0) 

The proposer sets this field from its own local config (`args.builder_address`): [2](#0-1) 

`convert_to_sn_api_block_info` maps `init.builder` directly to `sequencer_address` in the `BlockInfo` passed to the batcher: [3](#0-2) 

This `BlockInfo` is forwarded to the batcher via `ValidateBlockInput` in `initiate_validation`: [4](#0-3) 

`is_proposal_init_valid` validates `height`, `l1_da_mode`, `l2_gas_price_fri`, `starknet_version`, `version_constant_commitment`, all four L1 gas price fields, and `fee_proposal_fri`. The `builder` field is never checked against any locally-configured value: [5](#0-4) 

The `ProposalInitValidation` struct, which carries all locally-derived reference values, has no `builder` field: [6](#0-5) 

The locally-configured `builder_address` is a static config parameter: [7](#0-6) 

Because both the proposer and the validator derive `partial_block_hash` from the same proposer-supplied `builder` address, the `ProposalFin` commitment comparison at line 244 passes: [8](#0-7) 

The `proposal_commitment_from` function that produces the final commitment also uses the proposer-supplied `fee_proposal_fri` from `init`, not any locally-derived value: [9](#0-8) 

### Impact Explanation

In Starknet, `sequencer_address` is the fee recipient for every transaction in the block. A malicious proposer that sets `builder = ATTACKER_ADDRESS` causes:

1. **Wrong fee recipient**: All transaction fees in the block are transferred to `ATTACKER_ADDRESS` instead of the legitimate sequencer address. This is a direct economic theft of block revenue.
2. **Wrong block header**: The committed block header carries `ATTACKER_ADDRESS` as `sequencer_address`, corrupting the authoritative on-chain record.
3. **Wrong block hash**: Because `sequencer_address` is a component of the Starknet block hash, the committed block hash is computed over the attacker-supplied address, not the legitimate one.

This matches the **Critical** impact category: "Incorrect fee, gas, bouncer, resource accounting, refund, balance, or L1 gas price effect with economic impact."

### Likelihood Explanation

- Requires the attacker to be the designated proposer for a round, which is a normal role in a decentralized Tendermint validator set.
- No special privileges beyond being a selected proposer are needed.
- The attack is a single-field substitution in `ProposalInit`; no cryptographic forgery is required.
- The current deployment uses `assume_no_malicious_validators: true`, which lowers the quorum threshold and makes a single malicious proposer sufficient to commit the block. [10](#0-9) 

### Recommendation

Add `builder` to `ProposalInitValidation` and check it in `is_proposal_init_valid`:

```rust
// In ProposalInitValidation:
pub builder_address: ContractAddress,

// In is_proposal_init_valid:
if init_proposed.builder != proposal_init_validation.builder_address {
    return Err(ValidateProposalError::InvalidProposalInit(
        init_proposed.clone(),
        proposal_init_validation.clone(),
        format!(
            "builder mismatch: expected={:?}, proposed={:?}",
            proposal_init_validation.builder_address, init_proposed.builder
        ),
    ));
}
```

The `builder_address` should be populated from `self.builder_address` (the static config) when constructing `ProposalInitValidation` in `validate_proposal` / `validate_current_round_proposal`. [11](#0-10) 

### Proof of Concept

```
# Attacker is the designated proposer for round R at height H.
# Attacker modifies ProposalInit before broadcasting:

ProposalInit {
    height: H,
    round: R,
    proposer: ATTACKER_VALIDATOR_ADDRESS,
    builder: ATTACKER_FEE_RECIPIENT,   # <-- substituted; normally = legitimate_builder_address
    timestamp: <valid>,
    l1_da_mode: <matches validator config>,
    l2_gas_price_fri: <matches validator config>,
    l1_gas_price_fri: <within margin>,
    l1_data_gas_price_fri: <within margin>,
    l1_gas_price_wei: <within margin>,
    l1_data_gas_price_wei: <within margin>,
    starknet_version: LATEST,
    version_constant_commitment: <sentinel>,
    fee_proposal_fri: <within fee_actual bounds>,
}
```

1. Validator receives `ProposalInit` with `builder = ATTACKER_FEE_RECIPIENT`.
2. `is_proposal_init_valid` passes — `builder` is never checked.
3. `initiate_validation` calls `convert_to_sn_api_block_info(init)`, producing `BlockInfo { sequencer_address: ATTACKER_FEE_RECIPIENT, ... }`.
4. Batcher executes all transactions with `sequencer_address = ATTACKER_FEE_RECIPIENT`; every fee transfer credits `ATTACKER_FEE_RECIPIENT`.
5. Batcher returns `partial_block_hash` computed over `ATTACKER_FEE_RECIPIENT`.
6. Proposer's batcher computed the same `partial_block_hash` (same `builder`), so `ProposalFin.proposal_commitment == batcher_block_commitment` — the mismatch check at line 244 passes.
7. Block is committed; `ATTACKER_FEE_RECIPIENT` has received all fees; the on-chain block header records `ATTACKER_FEE_RECIPIENT` as sequencer.

#[instrument(level = "warn", skip_all, fields(?proposal_init_validation, ?init_proposed))]
async fn is_proposal_init_valid(
    proposal_init_validation: &ProposalInitValidation,
    init_proposed: &ProposalInit,
    clock: &dyn Clock,
    l1_gas_price_provider: Arc<dyn L1GasPriceProviderClient>,
    gas_price_params: &GasPriceParams,
) -> ValidateProposalResult<()> {
    let now: u64 = clock.unix_now();
    let last_block_timestamp =
        proposal_init_validation.previous_proposal_init.as_ref().map_or(0, |info| info.timestamp);
    if init_proposed.timestamp < last_block_timestamp {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "Timestamp is too old: last_block_timestamp={}, proposed={}",
                last_block_timestamp, init_proposed.timestamp
            ),
        ));
    }
    if init_proposed.timestamp > now + proposal_init_validation.block_timestamp_window_seconds {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "Timestamp is in the future: now={}, block_timestamp_window_seconds={}, \
                 proposed={}",
                now,
                proposal_init_validation.block_timestamp_window_seconds,
                init_proposed.timestamp
            ),
        ));
    }
    if init_proposed.starknet_version != proposal_init_validation.starknet_version {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "starknet_version mismatch: expected={:?}, proposed={:?}",
                proposal_init_validation.starknet_version, init_proposed.starknet_version
            ),
        ));
    }
    // `version_constant_commitment` is proposer-supplied (network-derived). It is not yet a real
    // commitment (see `expected_version_constant_commitment`): the only valid value is the
    // sentinel, so reject anything else. Enforcing the same value the proposer emits keeps the two
    // sides in lockstep, so a real value cannot ship on one side without the other.
    let expected_commitment = expected_version_constant_commitment();
    if init_proposed.version_constant_commitment != expected_commitment {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "version_constant_commitment mismatch: expected={expected_commitment:?}, \
                 proposed={:?}",
                init_proposed.version_constant_commitment
            ),
        ));
    }
    if !(init_proposed.height == proposal_init_validation.height
        && init_proposed.l1_da_mode == proposal_init_validation.l1_da_mode
        && init_proposed.l2_gas_price_fri == proposal_init_validation.l2_gas_price_fri)
    {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            "ProposalInit validation failed".to_string(),
        ));
    }
    let (l1_gas_prices_fri, l1_gas_prices_wei) = get_l1_prices_in_fri_and_wei(
        l1_gas_price_provider,
        init_proposed.timestamp,
        proposal_init_validation.previous_proposal_init.as_ref(),
        gas_price_params,
    )
    .await;
    let l1_gas_price_margin_percent =
        VersionedConstants::latest_constants().l1_gas_price_margin_percent.into();
    debug!("L1 price info: fri={l1_gas_prices_fri:?}, wei={l1_gas_prices_wei:?}");

    let l1_gas_price_fri = l1_gas_prices_fri.l1_gas_price;
    let l1_data_gas_price_fri = l1_gas_prices_fri.l1_data_gas_price;
    let l1_gas_price_wei = l1_gas_prices_wei.l1_gas_price;
    let l1_data_gas_price_wei = l1_gas_prices_wei.l1_data_gas_price;
    let l1_gas_price_fri_proposed = init_proposed.l1_gas_price_fri;
    let l1_data_gas_price_fri_proposed = init_proposed.l1_data_gas_price_fri;
    let l1_gas_price_wei_proposed = init_proposed.l1_gas_price_wei;
    let l1_data_gas_price_wei_proposed = init_proposed.l1_data_gas_price_wei;

    if !(within_margin(l1_gas_price_fri_proposed, l1_gas_price_fri, l1_gas_price_margin_percent)
        && within_margin(
            l1_data_gas_price_fri_proposed,
            l1_data_gas_price_fri,
            l1_gas_price_margin_percent,
        )
        && within_margin(l1_gas_price_wei_proposed, l1_gas_price_wei, l1_gas_price_margin_percent)
        && within_margin(
            l1_data_gas_price_wei_proposed,
            l1_data_gas_price_wei,
            l1_gas_price_margin_percent,
        ))
    {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "L1 gas price mismatch: expected L1 gas price FRI={l1_gas_price_fri}, \
                 proposed={l1_gas_price_fri_proposed}, expected L1 data gas price \
                 FRI={l1_data_gas_price_fri}, proposed={l1_data_gas_price_fri_proposed}, expected \
                 L1 gas price WEI={l1_gas_price_wei}, proposed={l1_gas_price_wei_proposed}, \
                 expected L1 data gas price WEI={l1_data_gas_price_wei}, \
                 proposed={l1_data_gas_price_wei_proposed}, \
                 l1_gas_price_margin_percent={l1_gas_price_margin_percent}"
            ),
        ));
    }

    // fee_proposal is required iff Starknet version >= V0_14_3.
    let fee_proposal_required = init_proposed.starknet_version >= StarknetVersion::V0_14_3;
    match (init_proposed.fee_proposal_fri, fee_proposal_required) {
        (Some(_), false) => {
            return Err(ValidateProposalError::InvalidProposalInit(
                init_proposed.clone(),
                proposal_init_validation.clone(),
                format!(
                    "fee_proposal must be absent before V0_14_3, got Some at version {}",
                    init_proposed.starknet_version
                ),
            ));
        }
        (None, true) => {
            return Err(ValidateProposalError::InvalidProposalInit(
                init_proposed.clone(),
                proposal_init_validation.clone(),
                format!(
                    "fee_proposal is required at V0_14_3+, got None at version {}",
                    init_proposed.starknet_version
                ),
            ));
        }
        _ => {}
    }

    // Validate fee_proposal is within the configured margin of fee_actual.
    // During initiation (fee_actual is None, <window_size blocks), bounds are not enforced.
    if let (Some(fee_actual), Some(fee_proposal)) =
        (proposal_init_validation.fee_actual, init_proposed.fee_proposal_fri)
    {
        let (lower_bound, upper_bound) = fee_proposal_bounds(
            fee_actual,
            VersionedConstants::latest_constants().fee_proposal_margin_ppt,
        );
        if fee_proposal.0 < lower_bound || fee_proposal.0 > upper_bound {
            return Err(ValidateProposalError::InvalidProposalInit(
                init_proposed.clone(),
                proposal_init_validation.clone(),
                format!(
                    "Fee proposal out of bounds: fee_actual={}, fee_proposal={}, allowed \
                     range=[{lower_bound}, {upper_bound}]",
                    fee_actual.0, fee_proposal.0
                ),
            ));
        }
    }

    Ok(())
}
