### Title
Silent L1 Handler Transaction Drop via `unwrap_or_default` in `ProposeTransactionProvider::get_l1_handler_txs` — (File: crates/apollo_batcher/src/transaction_provider.rs)

### Summary

`ProposeTransactionProvider::get_l1_handler_txs` calls the L1 events provider and silently discards any error via `.unwrap_or_default()`, returning an empty transaction list. The caller interprets an empty list as "no more L1 handler transactions available," immediately switches to the mempool phase, and builds the block without any L1 handler transactions. The resulting block is committed with a state root that omits all L1→L2 messages that were pending at that height.
