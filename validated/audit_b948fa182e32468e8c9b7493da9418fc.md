### Title
Unvalidated `ProposalInit.builder` Field Allows Malicious Proposer to Redirect All Block Transaction Fees — (`File: crates/apollo_consensus_orchestrator/src/validate_proposal.rs`)

---

### Summary

`is_proposal_init_valid` validates every security-sensitive field of `ProposalInit` received from a peer proposer **except** `builder`. `builder` is mapped verbatim to `sequencer_address` in `BlockInfo` by `convert_to_sn_api_block_info`, making it the fee-recipient address for every transaction executed in the block. Because no validator checks this field, any consensus-round proposer can set `builder` to an arbitrary address, redirecting the entire block's transaction fees to that address. All honest validators will accept and vote for the proposal, and the block is committed with the attacker-controlled `sequencer_address`.

---

### Finding Description

`ProposalInit` carries a `builder` field — the address of the entity that sequences the block: [1](#0-0) 

`convert_to_sn_api_block_info` maps this field directly to `sequencer_address` in the `BlockInfo` that is handed to the blockifier for execution: [2](#0-1) 

`sequencer_address` is the recipient of every fee transfer executed during the block. The blockifier's fee-charging logic transfers `actual_fee` to `block_context.block_info.sequencer_address` for every transaction.

`is_proposal_init_valid` — the sole gate that validates a received `ProposalInit` before the batcher is asked to execute the block — checks `timestamp`, `starknet_version`, `version_constant_commitment`, `height`, `l1_da_mode`, `l2_gas_price_fri`, all four L1 gas prices, and `fee_proposal_fri`. It never inspects `builder`: [3](#0-2) 

`ProposalInitValidation`, the struct that carries the locally-trusted reference values used for comparison, has no `builder_address` field at all: [4](#0-3) 

The proposer sets `builder` from its own static config (`builder_address`): [5](#0-4) 

A malicious proposer can substitute any address here. Because `is_proposal_init_valid` never compares `init_proposed.builder` against a locally-trusted value, the substitution passes validation silently. The tampered `init` is then forwarded to `initiate_validation`, which calls `convert_to_sn_api_block_info(init)` and passes the resulting `BlockInfo` (with the attacker's address as `sequencer_address`) to the batcher: [6](#0-5) 

Both the proposer's batcher and every validator's batcher execute the block with the same `init.builder`, so they produce identical `partial_block_hash` values. The `ProposalCommitment` matches on all sides, consensus reaches quorum, and the block is committed with the attacker-controlled `sequencer_address`.

After decision, `finalize_decision` calls `convert_to_sn_api_block_info(init)` again to produce the `cende_block_info` that is written to state-sync and the Aerospike blob: [7](#0-6) 

The committed block header permanently records the attacker's address as `sequencer_address`.

---

### Impact Explanation

**Critical — Incorrect fee/balance effect with economic impact.**

Every transaction fee in the block is transferred to the attacker-supplied `builder` address instead of the legitimate sequencer. Because the Starknet block hash includes `sequencer_address`, the committed block hash also encodes the attacker's address, making the corruption permanent and verifiable on-chain. The legitimate sequencer receives zero fees for the entire block.

---

### Likelihood Explanation

**Medium.** The attacker must be a registered consensus validator and must win the proposer slot for at least one round. In a Tendermint/BFT rotation, every validator eventually becomes proposer. No other precondition is required: the attack requires only crafting a `ProposalInit` with a modified `builder` field, which is trivially achievable by any validator node operator.

---

### Recommendation

Add a `builder_address: ContractAddress` field to `ProposalInitValidation` and populate it from the locally-configured `builder_address` when constructing the struct in `validate_proposal` (in `sequencer_consensus_context.rs`). Then add a check inside `is_proposal_init_valid`:

```rust
if init_proposed.builder != proposal_init_validation.builder_address {
    return Err(ValidateProposalError::InvalidProposalInit(
        init_proposed.clone(),
        proposal_init_validation.clone(),
        format!(
            "builder mismatch: expected={:?}, proposed={:?}",
            proposal_init_validation.builder_address, init_proposed.builder
        ),
    ));
}
```

This mirrors the pattern already used for `height`, `l1_da_mode`, and `l2_gas_price_fri` — all of which are validated by comparing the proposer-supplied value against a locally-trusted reference stored in `ProposalInitValidation`.

---

### Proof of Concept

1. Attacker operates a validator node and waits for (or triggers via timeout) a round in which it is the designated proposer.
2. In `initiate_build`, the attacker sets `builder: attacker_controlled_address` instead of `self.config.static_config.builder_address`.
3. The crafted `ProposalInit` is broadcast to all validators.
4. Each validator calls `validate_proposal` → `is_proposal_init_valid`. The function checks `timestamp`, `starknet_version`, `version_constant_commitment`, `height`, `l1_da_mode`, `l2_gas_price_fri`, L1 gas prices, and `fee_proposal_fri` — all of which are legitimate. `builder` is never read. Validation returns `Ok(())`.
5. `initiate_validation` calls `convert_to_sn_api_block_info(init)`, producing `BlockInfo { sequencer_address: attacker_controlled_address, … }`, and passes it to the batcher via `validate_block`.
6. The batcher executes every transaction with `sequencer_address = attacker_controlled_address`. All fee transfers go to the attacker.
7. Both the proposer's batcher and every validator's batcher use the same `init.builder`, so `partial_block_hash` is identical on all sides. `ProposalCommitment` matches; consensus reaches 2/3+ quorum.
8. `decision_reached` is called; `finalize_decision` calls `convert_to_sn_api_block_info(init)` again and writes the block header with `sequencer_address = attacker_controlled_address` to state-sync and the cende blob.
9. The block is permanently committed. The legitimate sequencer receives no fees; the attacker's address holds the entire block's fee revenue.

#[instrument(level = "warn", skip_all, fields(?proposal_init_validation, ?init_proposed))]
async fn is_proposal_init_valid(
    proposal_init_validation: &ProposalInitValidation,
    init_proposed: &ProposalInit,
    clock: &dyn Clock,
    l1_gas_price_provider: Arc<dyn L1GasPriceProviderClient>,
    gas_price_params: &GasPriceParams,
) -> ValidateProposalResult<()> {
    let now: u64 = clock.unix_now();
    let last_block_timestamp =
        proposal_init_validation.previous_proposal_init.as_ref().map_or(0, |info| info.timestamp);
    if init_proposed.timestamp < last_block_timestamp {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "Timestamp is too old: last_block_timestamp={}, proposed={}",
                last_block_timestamp, init_proposed.timestamp
            ),
        ));
    }
    if init_proposed.timestamp > now + proposal_init_validation.block_timestamp_window_seconds {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "Timestamp is in the future: now={}, block_timestamp_window_seconds={}, \
                 proposed={}",
                now,
                proposal_init_validation.block_timestamp_window_seconds,
                init_proposed.timestamp
            ),
        ));
    }
    if init_proposed.starknet_version != proposal_init_validation.starknet_version {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "starknet_version mismatch: expected={:?}, proposed={:?}",
                proposal_init_validation.starknet_version, init_proposed.starknet_version
            ),
        ));
    }
    // `version_constant_commitment` is proposer-supplied (network-derived). It is not yet a real
    // commitment (see `expected_version_constant_commitment`): the only valid value is the
    // sentinel, so reject anything else. Enforcing the same value the proposer emits keeps the two
    // sides in lockstep, so a real value cannot ship on one side without the other.
    let expected_commitment = expected_version_constant_commitment();
    if init_proposed.version_constant_commitment != expected_commitment {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "version_constant_commitment mismatch: expected={expected_commitment:?}, \
                 proposed={:?}",
                init_proposed.version_constant_commitment
            ),
        ));
    }
    if !(init_proposed.height == proposal_init_validation.height
        && init_proposed.l1_da_mode == proposal_init_validation.l1_da_mode
        && init_proposed.l2_gas_price_fri == proposal_init_validation.l2_gas_price_fri)
    {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            "ProposalInit validation failed".to_string(),
        ));
    }
    let (l1_gas_prices_fri, l1_gas_prices_wei) = get_l1_prices_in_fri_and_wei(
        l1_gas_price_provider,
        init_proposed.timestamp,
        proposal_init_validation.previous_proposal_init.as_ref(),
        gas_price_params,
    )
    .await;
    let l1_gas_price_margin_percent =
        VersionedConstants::latest_constants().l1_gas_price_margin_percent.into();
    debug!("L1 price info: fri={l1_gas_prices_fri:?}, wei={l1_gas_prices_wei:?}");

    let l1_gas_price_fri = l1_gas_prices_fri.l1_gas_price;
    let l1_data_gas_price_fri = l1_gas_prices_fri.l1_data_gas_price;
    let l1_gas_price_wei = l1_gas_prices_wei.l1_gas_price;
    let l1_data_gas_price_wei = l1_gas_prices_wei.l1_data_gas_price;
    let l1_gas_price_fri_proposed = init_proposed.l1_gas_price_fri;
    let l1_data_gas_price_fri_proposed = init_proposed.l1_data_gas_price_fri;
    let l1_gas_price_wei_proposed = init_proposed.l1_gas_price_wei;
    let l1_data_gas_price_wei_proposed = init_proposed.l1_data_gas_price_wei;

    if !(within_margin(l1_gas_price_fri_proposed, l1_gas_price_fri, l1_gas_price_margin_percent)
        && within_margin(
            l1_data_gas_price_fri_proposed,
            l1_data_gas_price_fri,
            l1_gas_price_margin_percent,
        )
        && within_margin(l1_gas_price_wei_proposed, l1_gas_price_wei, l1_gas_price_margin_percent)
        && within_margin(
            l1_data_gas_price_wei_proposed,
            l1_data_gas_price_wei,
            l1_gas_price_margin_percent,
        ))
    {
        return Err(ValidateProposalError::InvalidProposalInit(
            init_proposed.clone(),
            proposal_init_validation.clone(),
            format!(
                "L1 gas price mismatch: expected L1 gas price FRI={l1_gas_price_fri}, \
                 proposed={l1_gas_price_fri_proposed}, expected L1 data gas price \
                 FRI={l1_data_gas_price_fri}, proposed={l1_data_gas_price_fri_proposed}, expected \
                 L1 gas price WEI={l1_gas_price_wei}, proposed={l1_gas_price_wei_proposed}, \
                 expected L1 data gas price WEI={l1_data_gas_price_wei}, \
                 proposed={l1_data_gas_price_wei_proposed}, \
                 l1_gas_price_margin_percent={l1_gas_price_margin_percent}"
            ),
        ));
    }

    // fee_proposal is required iff Starknet version >= V0_14_3.
    let fee_proposal_required = init_proposed.starknet_version >= StarknetVersion::V0_14_3;
    match (init_proposed.fee_proposal_fri, fee_proposal_required) {
        (Some(_), false) => {
            return Err(ValidateProposalError::InvalidProposalInit(
                init_proposed.clone(),
                proposal_init_validation.clone(),
                format!(
                    "fee_proposal must be absent before V0_14_3, got Some at version {}",
                    init_proposed.starknet_version
                ),
            ));
        }
        (None, true) => {
            return Err(ValidateProposalError::InvalidProposalInit(
                init_proposed.clone(),
                proposal_init_validation.clone(),
                format!(
                    "fee_proposal is required at V0_14_3+, got None at version {}",
                    init_proposed.starknet_version
                ),
            ));
        }
        _ => {}
    }

    // Validate fee_proposal is within the configured margin of fee_actual.
    // During initiation (fee_actual is None, <window_size blocks), bounds are not enforced.
    if let (Some(fee_actual), Some(fee_proposal)) =
        (proposal_init_validation.fee_actual, init_proposed.fee_proposal_fri)
    {
        let (lower_bound, upper_bound) = fee_proposal_bounds(
            fee_actual,
            VersionedConstants::latest_constants().fee_proposal_margin_ppt,
        );
        if fee_proposal.0 < lower_bound || fee_proposal.0 > upper_bound {
            return Err(ValidateProposalError::InvalidProposalInit(
                init_proposed.clone(),
                proposal_init_validation.clone(),
                format!(
                    "Fee proposal out of bounds: fee_actual={}, fee_proposal={}, allowed \
                     range=[{lower_bound}, {upper_bound}]",
                    fee_actual.0, fee_proposal.0
                ),
            ));
        }
    }

    Ok(())
}
