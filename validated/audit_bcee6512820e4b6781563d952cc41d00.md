### Title
Pool Wallet `join_pool()` Allows Pool Operator to Divert Unclaimed Self-Pooling XCH Rewards After State Transition - (File: `chia/pools/pool_wallet.py`)

### Summary

When a farmer transitions from `SELF_POOLING` to `FARMING_TO_POOL` via `join_pool()`, any coinbase rewards already sitting at the fixed `p2_singleton_puzzle_hash` can be permissionlessly absorbed by the pool operator after the state transition, with the absorbed amount routed to the pool's `target_puzzle_hash` instead of the farmer's address. The `join_pool()` function performs no check for, and makes no attempt to claim, outstanding p2_singleton rewards before committing the state change.

### Finding Description

**Root cause — `join_pool()` has no unclaimed-reward guard**

`PoolWallet.join_pool()` only checks for an unconfirmed transaction before proceeding:

```python
if await self.have_unconfirmed_transaction():
    raise ValueError(...)
```

There is no check for unclaimed coinbase rewards at the `p2_singleton_puzzle_hash`. [1](#0-0) 

**The p2_singleton puzzle hash is state-independent**

`launcher_id_to_p2_puzzle_hash` depends only on `launcher_id`, `delay_time`, and `delay_ph` — not on the singleton's current inner puzzle. Rewards farmed while self-pooling land at the same address that is watched after joining a pool. [2](#0-1) 

**`pool_state_to_inner_puzzle` maps SELF_POOLING and FARMING_TO_POOL to different inner puzzles with different `target_puzzle_hash` values**

- `SELF_POOLING` → `POOL_WAITING_ROOM_MOD` with farmer's own address as `target_puzzle_hash`
- `FARMING_TO_POOL` → `POOL_MEMBER_MOD` with pool's address as `target_puzzle_hash` [3](#0-2) 

**`create_absorb_spend` is permissionless and height-agnostic**

`create_absorb_spend` constructs the reward coin purely from `height` and `genesis_challenge`. It requires no signature (fee=0 path) and does not verify that the reward was farmed after the current state was entered. Any party who knows the block height can submit an absorb spend. [4](#0-3) 

After the state transition to `FARMING_TO_POOL`, the absorb spend routes the reward to `current_state.current.target_puzzle_hash` — the pool's address:

```python
puzzle_hash=current_state.current.target_puzzle_hash,
``` [5](#0-4) 

**Attack sequence**

1. Farmer is in `SELF_POOLING` state; one or more coinbase rewards sit unabsorbed at `p2_singleton_puzzle_hash` (farmed at block heights H₁, H₂, …).
2. Farmer calls `join_pool()`. The singleton transitions to `FARMING_TO_POOL` with `target_puzzle_hash` = pool's address.
3. Pool operator (or any unprivileged party) calls `create_absorb_spend(last_coin_spend, FARMING_TO_POOL_state, launcher_coin, H₁, ...)`.
4. The spend is valid on-chain (no signature required, p2_singleton puzzle hash unchanged). The reward is sent to the pool's address.
5. Farmer permanently loses the self-pooling XCH reward.

**Why the farmer cannot fully protect themselves**

The farmer can call `claim_pool_rewards` before `join_pool`, but both operations require no unconfirmed transactions, so they cannot be made atomic. A new block reward can arrive at `p2_singleton_puzzle_hash` in the interval between the claim confirmation and the join confirmation. The pool operator can observe the farmer's pending `join_pool` transaction and time an absorb spend to land in the same block or the next block after the state transition confirms.

The `claim_pool_rewards` function itself also only processes coins tracked as farming rewards in the local tx store; coins that arrive after the last sync are silently skipped. [6](#0-5) 

### Impact Explanation

A malicious pool operator can permanently divert the farmer's self-pooling XCH coinbase rewards (7/8 of block reward per block) to the pool's own address. This is an unauthorized payout redirection of XCH affecting the pool wallet singleton. The farmer has no on-chain recourse once the absorb spend is confirmed.

### Likelihood Explanation

Any farmer who has farmed at least one block while self-pooling and then joins a pool is exposed. The pool operator is an unprivileged party relative to the farmer's wallet and can observe the blockchain to identify unclaimed rewards. No key material, admin access, or cryptographic break is required.

### Recommendation

`join_pool()` should enumerate unclaimed coinbase rewards at `p2_singleton_puzzle_hash` and either:
- Refuse to proceed if unclaimed rewards exist (forcing the farmer to absorb them first), or
- Atomically include absorb spends for all outstanding rewards in the same spend bundle as the state-transition spend, routing them to the farmer's current `target_puzzle_hash` before the inner puzzle changes.

### Proof of Concept

```
State: SELF_POOLING, target_puzzle_hash = FARMER_PH
Block H farmed → reward coin at p2_singleton_puzzle_hash (amount = calculate_pool_reward(H))

Farmer calls join_pool(target_puzzle_hash=POOL_PH)
  → join_pool() checks only have_unconfirmed_transaction() — passes
  → singleton transitions to FARMING_TO_POOL, target_puzzle_hash = POOL_PH

Pool operator calls create_absorb_spend(last_spend, FARMING_TO_POOL_state, launcher, H, ...)
  → inner_puzzle = POOL_MEMBER_MOD (target = POOL_PH)
  → reward_coin = Coin(pool_parent_id(H, genesis), p2_singleton_ph, calculate_pool_reward(H))
  → p2_singleton_solution = [POOL_MEMBER_inner_hash, reward_coin.name()]
  → SpendBundle submitted with G2Element() (no signature needed)
  → reward routed to POOL_PH ✓

Farmer's self-pooling reward is permanently diverted.
``` [7](#0-6) [4](#0-3) [3](#0-2)

            raise ValueError(
                "Cannot join pool due to unconfirmed transaction. If this is stuck, delete the unconfirmed transaction."
            )

        current_state: PoolWalletInfo = await self.get_current_state()

        total_fee = fee
        if current_state.current == target_state:
            self.target_state = None
            msg = f"Asked to change to current state. Target = {target_state}"
            self.log.info(msg)
            raise ValueError(msg)
        elif current_state.current.state in {SELF_POOLING.value, LEAVING_POOL.value}:
            total_fee = fee
        elif current_state.current.state == FARMING_TO_POOL.value:
            total_fee = uint64(fee * 2)

        if self.target_state is not None:
            raise ValueError(
                f"Cannot change to state {target_state} when already having target state: {self.target_state}"
            )
        PoolWallet._verify_initial_target_state(target_state)
        if current_state.current.state == LEAVING_POOL.value:
            history: list[tuple[uint32, CoinSpend]] = await self.get_spend_history()

            if (
                await self.wallet_state_manager.blockchain.get_finished_sync_up_to()

        self.next_transaction_fee = fee

def create_absorb_spend(
    last_coin_spend: CoinSpend,
    current_state: PoolState,
    launcher_coin: Coin,
    height: uint32,
    genesis_challenge: bytes32,
    delay_time: uint64,
    delay_ph: bytes32,
) -> list[CoinSpend]:
    inner_puzzle: Program = pool_state_to_inner_puzzle(
        current_state, launcher_coin.name(), genesis_challenge, delay_time, delay_ph
    )
    reward_amount: uint64 = calculate_pool_reward(height)
    if is_pool_member_inner_puzzle(inner_puzzle):
        # inner sol is (spend_type, pool_reward_amount, pool_reward_height, extra_data)
        inner_sol: Program = Program.to([reward_amount, height])
    elif is_pool_waitingroom_inner_puzzle(inner_puzzle):
        # inner sol is (spend_type, destination_puzhash, pool_reward_amount, pool_reward_height, extra_data)
        inner_sol = Program.to([0, reward_amount, height])
    else:
        raise ValueError
    # full sol = (parent_info, my_amount, inner_solution)
    coin: Coin | None = get_most_recent_singleton_coin_from_coin_spend(last_coin_spend)
    assert coin is not None

    if coin.parent_coin_info == launcher_coin.name():
        parent_info: Program = Program.to([launcher_coin.parent_coin_info, launcher_coin.amount])
    else:
        p = Program.from_bytes(bytes(last_coin_spend.puzzle_reveal))
        last_coin_spend_inner_puzzle: Program | None = get_inner_puzzle_from_puzzle(p)
        assert last_coin_spend_inner_puzzle is not None
        parent_info = Program.to(
            [
                last_coin_spend.coin.parent_coin_info,
                last_coin_spend_inner_puzzle.get_tree_hash(),
                last_coin_spend.coin.amount,
            ]
        )
    full_solution: SerializedProgram = SerializedProgram.to([parent_info, last_coin_spend.coin.amount, inner_sol])
    full_puzzle: SerializedProgram = create_full_puzzle(inner_puzzle, launcher_coin.name()).to_serialized()
    assert coin.puzzle_hash == full_puzzle.get_tree_hash()

    reward_parent: bytes32 = pool_parent_id(height, genesis_challenge)
    p2_singleton_puzzle = create_p2_singleton_puzzle(
        SINGLETON_MOD_HASH, launcher_coin.name(), delay_time, delay_ph
    ).to_serialized()
    reward_coin: Coin = Coin(reward_parent, p2_singleton_puzzle.get_tree_hash(), reward_amount)
    p2_singleton_solution = SerializedProgram.to([inner_puzzle.get_tree_hash(), reward_coin.name()])
    assert p2_singleton_puzzle.get_tree_hash() == reward_coin.puzzle_hash
    assert full_puzzle.get_tree_hash() == coin.puzzle_hash
    assert get_inner_puzzle_from_puzzle(Program.from_bytes(bytes(full_puzzle))) is not None

    coin_spends = [
        CoinSpend(coin, full_puzzle, full_solution),
        CoinSpend(reward_coin, p2_singleton_puzzle, p2_singleton_solution),
    ]
    return coin_spends

def pool_state_to_inner_puzzle(
    pool_state: PoolState, launcher_id: bytes32, genesis_challenge: bytes32, delay_time: uint64, delay_ph: bytes32
) -> Program:
    escaping_inner_puzzle: Program = create_waiting_room_inner_puzzle(
        pool_state.target_puzzle_hash,
        pool_state.relative_lock_height,
        pool_state.owner_pubkey,
        launcher_id,
        genesis_challenge,
        delay_time,
        delay_ph,
    )
    if pool_state.state in {LEAVING_POOL.value, SELF_POOLING.value}:
        return escaping_inner_puzzle
    else:
        return create_pooling_inner_puzzle(
            pool_state.target_puzzle_hash,
            escaping_inner_puzzle.get_tree_hash(),
            pool_state.owner_pubkey,
            launcher_id,
            genesis_challenge,
            delay_time,
            delay_ph,
        )
